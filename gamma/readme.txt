Группа: КБ-501
Задача: 12 - Гаммирование

Описание:
    Исходный файл перерабатывается в уравновешенную гамму алгоритмом шифрования следующим образом:
        1) Исходный текст шифруется.
        2) Пока не выполнено условие уравновешенности битов выбирается блок с худшей уравновешенностью того же типа,
         что и неуравновешенность текста в целом и снова шифруется.
Плюшки:
    1) Размер вырабатываемой гаммы ограничен лишь возможностями системы. Все операции чтения, шифрования, записи осуществляются по блокам.
    2) Madryga прекрасно вырабатывает гамму, что бы блоки начали шифроваться по нескольку раз,
     приходится выставлять максимально допустимое отклонение в 0.0015 (против 0.1 в условии задачи)

Описание файлов:
    gamma.py - содержит точку входа в программу, отвечает за разбор аргументов
    cio.pxd - заголовочный файл для cio.pyx
    cio.pyx - cython обертка C-ных функций работы с файлами (дает выигрыш относительно стандартных для python методов чтения/записи в файл) 
    file_encriptor.pyx - cython скрипт осуществляет блочное чтение с использованием cio.pyx, и гаммирование файла, используя алгоритм шифрования  
    madryga.pyx - cython обертка над функциями, описанными в madryga_cipher.h
    madryga_cipher.c - реализация функций описанных в madryga_cipher.h на C
    madryga_cipher.h - заголовочный файл с декларацией методов шифрования/расшифрования
    setup.py - автоматизированная сборка cython .pyx файлов в .pyd модули
    build.bat - bat скрипт для запуска сборки.
    cio.pyd, madryga.pyd, file_encriptor.pyd - скомпилированные модули

Запуск:
    Программа реализована под Python 3.4.1
Параметры запуска:
    gamma.py 
        -k, --key - задает ключ шифрования в 16-ричном виде
        -f, --file - задает файл для переработки в гамму
        -s, --blockSize - размер блока (должен быть степень 2-ки)
        -i, --iterations - указывает число итераций внешнего цикла
        -d, --max_deviation - задает максимальное отклонение в долях от состояния уравновешенности

    Пример: python34 gamma.py -f 1.txt -k 0 -s 16 --iterations=5 -d 0.0004
