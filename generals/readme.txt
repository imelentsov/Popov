Задача: 8 - Византийские генералы

Описание:
    Программа работает в двух режимах:
        1) генерирует файл пересылок векторов (последний массив массивов массивов - есть синхронизированные вектора) (работает для любых размеров задачи, при условии существовании решения, и возможности ее решить на системе - по памяти и времени. Пробовал для (85, 15) - работает меньше минуты)
        2) в красивом виде выводит на консоль результаты раундов (работает до n = 18, красиво работает до 15)

Формат генерируемого файла:
    Задача Византийских генералов геннерируется в виде json файла, представляющего из себя массив раундов. 
    Каждый раунд представляет из себя массив, элементы которого есть массив знаний для каждого из лояльных генералов, элементы которого есть вектора:
        1-й вектор в массиве есть набор чисел полученных от других генералов (и свое число) для первого раунда,
          или результат голосования генерала по векторам полученным на предыдущем раунде для остальных раундов.
        Остальные вектора есть вектора полученные от других генералов
    В файл так же в том же формате записывается результат последнего голосования.   

Запуск:
    Программа реализована под Python 3.4.1
Параметры запуска:
    gamma.py
        -f, --file - Задает файл из которого будет прочитана(или записана с ключом -g) 
                последовательность пересылок векторов согласно протоколу согласования "Византийских генералов"
        -g - "n m" Указывает, что необходимо сгенерировать новую (n, m) задачу Византийских генералов. 
            Без этого параметра программа в красивом виде выводит на экран результаты каждого раунда, работает до n = 18.

    Пример: python34 generals.py -f 1.txt -g "13 4"
