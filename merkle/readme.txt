Задача: 7 - Широковещание

Описание:
    Широковещание реализовал на основе головоломки Меркле.
    Программа есть имитация взаимодействия клиента и сервера.
    Сервер генерирует набор зашифрованных слабым алгоритмом шифрования сообщений содержащих ключ и идентификатор сообщения, хранит сообщения в перемешенном порядке.
    Клиент произвольным образом выбирает ключевое сообщение, брутфорсом расшифровывает, получает ключ и передает серверу идентификатор сообщения.
    Сервер находит у себя по идентификатору нужное зашифрованное сообщение, расшифровывает брутфорсом сообщение, и тоже получает ключ. 
    Если ключи совпали клиент и сервер могут нормально общаться по криптостойкому алгоритму с согласованными ключами.

Описание файлов:
    merkle.py -  реализация головоломки Меркле
    madryga.pyx - cython обертка над функциями описанными в madryga_cipher.h
    madryga_cipher.cpp - реализация алгоритма шифрования на С++
    madryga_cipher.h - заголовочный файл с декларацией методов шифрования/раcшифрования
    setup.py - автоматизированная сборка cython .pyx файлов в .pyd модули
    build.bat - bat скрипт для запуска сборки
    madryga.pyd- скомпилированный модуль

Запуск:
    Программа реализована под Python 3.4.1
Параметры запуска: 
    Скрипт запускается без параметров
